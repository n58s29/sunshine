ℹ️ Mode d'emploi

Cet outil remplace les supports PPTX pour les sessions de sensibilisation à l'IA Générative. Un seul fichier HTML, zéro installation, qui couvre tout le cycle d'une session : préparation, animation et suivi.

Avant

Préparer la session

Ouvrir le panneau de configuration avec le bouton ✏️ et renseigner les informations du collectif, la date, le nombre de participants, l'email du commanditaire et le lien vers le questionnaire de satisfaction.

Activer ou désactiver les fonctionnalités GenAI pertinentes pour le métier concerné. Personnaliser le cas d'usage de chaque fonctionnalité avec un exemple concret parlant pour le collectif. Réordonner les fonctionnalités par glisser-déposer selon la priorité de la session.

Coller le lien de la démo GPT SNCF pour chaque fonctionnalité afin d'y accéder en un clic pendant la session.

Sauvegarder la configuration en JSON pour la réutiliser ou la partager avec un autre animateur.

Pendant

Animer la session

Projeter la page en plein écran. Chaque fonctionnalité apparaît avec son cas d'usage métier adapté au collectif.

Cliquer sur le bouton Démo d'une fonctionnalité pour ouvrir directement la conversation GPT SNCF correspondante et faire la démonstration en live.

Au fil de la session, cliquer sur le rond ✓ à droite de chaque fonctionnalité pour la marquer comme vue. Elle passe visuellement en mode atténué, ce qui permet de suivre la progression et de voir ce qu'il reste à montrer.

Après

Assurer le suivi

Cliquer sur le bouton 📧 Envoyer pour générer un fichier .eml. L'ouvrir avec Outlook : un mail HTML professionnel est prêt à envoyer au commanditaire avec le récapitulatif complet des fonctionnalités et cas d'usage abordés, le lien vers GPT SNCF, et le questionnaire de satisfaction mis en avant.

Exporter la configuration en JSON avec 💾 JSON pour archiver la session. Le fichier est nommé automatiquement avec la date et le collectif.
